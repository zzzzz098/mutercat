# Floating Hearts 💕

一个纯 HTML/CSS/JS 实现的浮动爱心动画效果，在渐变背景上飘落各种爱心 emoji。

## 效果预览

打开 `floating-hearts.html` 即可在浏览器中看到 80 个随机大小、随机速度的爱心从屏幕顶部飘落。

## 使用方法

1. 克隆仓库：
   ```bash
   git clone https://github.com/zzzzz098/mutercat.git
   ```
2. 直接用浏览器打开 `floating-hearts.html`

或者直接访问 GitHub Pages 预览。

## 自定义

- 修改 `EMOJIS` 数组（第 37 行）可以更换飘落的 emoji
- 修改 `COUNT`（第 38 行）可以调整爱心数量
- 修改 CSS 中的 `background`（第 14 行）可以更换背景渐变
